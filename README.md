Associations between in vitro, in vivo and in silico cell classes in mouse primary visual cortex (Wei et al, 2021)

Code base for mouse V1 Neurpixels EAP data analysis

* Method is presented in Wei et al. (2021): 

## Dependencies
 * Python 3.6+
 * AllenSDK
 * pandas
 * Seaborn
 * Matplotlib
 * scikit-learn
 * SciPy
 * Random
 * scikit_posthocs
